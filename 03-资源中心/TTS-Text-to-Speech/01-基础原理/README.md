# 基础原理

## 语音合成基本概念

### 核心组件
1. **文本分析器** - 文本预处理和特征提取
2. **声学模型** - 生成声学特征
3. **声码器** - 将声学特征转换为音频

### 技术路线
- **拼接式合成**: 预录制语音片段拼接
- **参数式合成**: 统计参数模型生成语音
- **端到端合成**: 深度学习直接生成音频

## 关键技术

### 声学特征
- MFCC (Mel频率倒谱系数)
- F0 (基频)
- 频谱包络

### 声码器类型
- 源-滤波器模型
- 神经声码器 (WaveNet, WaveRNN, HiFi-GAN)

## 数学基础

### 信号处理
- 傅里叶变换
- 梅尔尺度
- 线性预测编码
